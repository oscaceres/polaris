Centurion 28 Telescope Manual

28-inch Diameter Concave Mirror at Prime Focus

At the start of the night, complete the following steps:

  1. Check the weather (Clear Dark Sky, Clarity II, NOAA Civil Weather, Moon Phase/Illumination).
  2. Email Dr. Rich.
  3. Start cooling the CCD camera 1 hour before sunset.

Calibrations: Bias and Dark

  1. SkyRoof -> Open (Wait until the roof fully opens)
  2. Outlet Control -> Turn OFF IR Sensor (4) (Scope is now unparked)
  3. Open Sequence Generator Pro and PHD2 Guiding
  4. On SGPro:
  • Connect camera and equipment.
  • Set directory and create a new folder (Today’s date plus one: daymonthyear-filter [e.g. 09apr24-N2])
  • Deselect all targets (To clear targets from last night)
  • Reset and select bias-start and dark-start
  • File -> Apply profile to sequence -> Select camera profile -> ok
  • File -> Save sequence
  • Ensure the camera temperature is near 90% cooler power
  1. Start the sequence and inspect the frames (Refer to frames in previous folders) for any inconsistencies or signs of oversaturation.

Stage 2: Imaging Galaxies

  1. Deselect all targets or deselect bias-start and dark-start.
  2. Select the target pointings you want to observe (Above 45* in altitude; or above ~20* pointing South [Picture the telescope as it moves with respect to the walls and the roof])
  3. Adjust target end times using the Planning tool and Stellarium.
  4. Reset and select bias-end and dark-end.
  5. File -> Save sequence
  6. Double click or right click and open the ‘Powerup & Sync’ VBS script. [Opens TheSky6 application and turns ON the telescope mount]
  7. Start sequence (Observe the slewing of the telescope with Astroworks and the ASICAP boresight camera)

Shutdown procedure [Sunrise/Emergency]

  1. Turn on IR Sensor
  2. Power Down & Sync OR  manually move/slew the telescope to the parked position in the South using TheSky6 while monitoring the red circle on Astroworks. To do this,
  • Go to telescope -> motion controls
  • Slew to the parked position using the controls; select ‘move’ for smaller movements.
  • On SkyRoof, the green ‘scope is parked’ should replace the yellow ‘scope is unparked’
  • Go to Outlet Control and turn off Scope Mount (Telescope is disconnected and not free to move)
  1. If the telescope is parked and the scope mount is off, click ‘close’ on SkyRoof
  2. On SGPro, deselect all targets. Reset and select bias-end and dark-end. Make sure the CCD camera temperature is the same temperature it was overnight.
  3. When the roof is closed, run the sequence. If you already took end calibrations, you may close SGPro, PHD2 Guiding, and TheSky6.
  4. Inspect the data and upload to the shared Google Drive folder.
